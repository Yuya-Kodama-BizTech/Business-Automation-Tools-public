# VBA自動請求書発行ツール (VBA Invoice Generator)

## 📌 概要
このツールは、Excel上の請求データ（会社名、金額、明細など）を読み取り、指定された請求書テンプレートに自動で転記、PDFとして一括保存するVBAマクロです。
既存のExcel業務のフローを崩さず、最小限の導入コストで最大の効率化を実現します。

## 🚀 導入のメリット
- **劇的な時間短縮**: 
  手作業で1社あたり約5分かかっていた請求書作成・PDF保存作業を、マクロ実行により一瞬（数秒）で完了させます。
- **ヒューマンエラーの完全防止**: 
  データのコピー＆ペーストミスや、宛先間違いなどの手作業特有のミスを排除し、請求業務の信頼性を高めます。
- **一括管理・自動保存**: 
  出力されたPDFは、実行した日付とともに自動生成されたフォルダへ整理・保存されます。

## 🛠 使用技術
- **言語**: Excel VBA (Visual Basic for Applications)
- **機能**: 
  - オブジェクト操作（Worksheet, Range, Workbook）
  - PDFエクスポート機能 (`ExportAsFixedFormat`)
  - ファイル・フォルダ自動生成 (`Dir`, `MkDir`)

## 📝 注意事項
- **互換性**: Microsoft Excel 2010以降のバージョンで動作します。
- **セキュリティ**: 実行時には「マクロの有効化」が必要です。
