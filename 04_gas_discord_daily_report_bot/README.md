# Discord × GAS 社内報告活性化 Bot (Discord Daily Report Bot)

## 📌 概要
Google Apps Script (GAS) と Discord Webhook を連携させ、チーム内の「朝の予定宣言」と「夜の成果報告」を自動で促す Bot システムです。
日々の進捗管理を文化として定着させ、管理者とメンバー双方の負担を軽減することを目的としています。

## 🚀 導入のメリット
- **コミュニケーションコストの大幅削減**: 
  上長が一人ひとりに進捗を確認しに行く手間を省き、チーム全体で「誰が・何を・どこまで」進めているかをリアルタイムで共有。
- **社員の意識づけとタスクの明確化**: 
  朝夜の定点報告により、メンバー自身が「本日取り組むべきタスク」と「得られた成果」を意識する習慣が定着しました。
- **導入の容易さ**: 
  既存の Discord 環境と Google スプレッドシートがあれば、数分で導入・運用開始が可能です。

## 🛠 機能概要
- **朝の自動リマインド**: 
  毎朝、指定の時間に Discord 上で本日の予定入力を促すメッセージを送信。
- **夜の自動リマインド**: 
  退勤前の指定の時間に、成果報告と翌日のタスクの入力を促すメッセージを送信。
- **スプレッドシート連携（オプション）**: 
  投稿された内容をスプレッドシートに蓄積し、月次の業務レポート作成などに活用可能です。

## 🛠 使用技術
- **言語**: Google Apps Script (JavaScript)
- **プラットフォーム**: Discord (Webhook)
- **データベース**: Google Spreadsheets (ログ管理用)
