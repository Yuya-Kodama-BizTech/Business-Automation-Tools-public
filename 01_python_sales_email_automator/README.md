# 営業メール自動化ツール (Sales Email Automator)

## 📌 概要
このツールは、Excel形式の顧客リストから情報を自動で読み取り、**Gmail および Outlook** を介してパーソナライズされた営業メールを一括送信するPythonスクリプトです。
日々のルーチンワークを自動化し、戦略的な業務に集中できる時間を創出します。

## 🚀 導入のメリット
- **月間約30時間の削減**: 
  1件ずつ手作業で送信していた営業メール（1件あたり3分 × 月600件）を自動化。業務効率を劇的に向上させます。
- **マルチプラットフォーム対応**: 
  Gmail だけでなく、多くの企業で導入されている **Outlook (Microsoft 365)** にも対応。
- **ヒューマンエラーの排除**: 
  手動入力による宛先間違いや氏名の誤字、添付漏れなどのリスクをゼロにします。
- **一貫したコミュニケーション**: 
  定型文を活用しつつ、会社名や氏名を動的に埋め込むことで、丁寧かつスピーディなアプローチを可能にします。

## 🛠 使用技術
- **言語**: Python 3.x
- **ライブラリ**: 
  - `pandas`: Excelデータの読み込み・加工
  - `smtplib`: SMTPサーバーを介したメール送信プロトコルの制御
  - `email.mime`: MIME形式（マルチパートメール）の構築
- **インフラ/サービス**: 
  - **Gmail / Outlook (Microsoft 365)** SMTP Server
  - Excel (データソース)
  - Anaconda
  - jupterlab
