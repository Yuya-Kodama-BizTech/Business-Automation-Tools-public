# X (Twitter) 自動投稿システム (X Auto Poster with GAS)

## 📌 概要
Google Apps Script (GAS) と X API を連携させ、スプレッドシートから自動でコンテンツを投稿するシステムです。
単なる自動投稿に留まらず、広告運用や求人募集におけるインプレッション獲得を最大化するための機能を搭載しています。

## 🚀 実績と導入メリット
- **月間最大2,000万 Impression 獲得**: 
  戦略的なコンテンツ配信と24時間の自動運用により、圧倒的な認知拡大を実現。
- **24時間365日の完全自動運用**: 
  GASの時間ベーストリガーを活用し、深夜・早朝を含む最適なタイミング（ランダム設定可能）での投稿を継続します。
- **永久的なループ運用**: 
  用意した10種類以上のコンテンツを使い切った際、自動でリストをリフレッシュし、再度ランダムな順序で投稿を開始します。

## 🛠 技術的な工夫（bot判定対策）
- **透明文字（不可視文字）のランダム付与**: 
  投稿内容の末尾に、見た目には影響しない不可視文字（Zero Width Spaceなど）をランダムな数だけ挿入。同一内容の繰り返し投稿と判定されるのを防ぎます。
- **投稿時間の分散**: 
  GASのトリガーと独自のランダム遅延ロジックを組み合わせ、機械的な投稿パターンを回避しています。

## 🛠 使用技術
- **言語**: Google Apps Script (JavaScript)
- **API**: X API v2 (OAuth 1.0a / 2.0)
- **データベース**: Google Spreadsheets
- **実行環境**: Google Cloud (Serverless)

## 📝 注意事項
- **API制限**: X APIのレートリミット（1日あたりの投稿上限など）を考慮した設計が必要です。
- **運用ポリシー**: Xの利用規約を遵守し、スパム行為に当たらない健全な運用を前提としています。
