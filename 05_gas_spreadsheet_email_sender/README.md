# スプレッドシート連携 営業メール自動送信ツール (GAS Spreadsheet Email Sender)

## 📌 概要
Google Apps Script (GAS) を活用し、スプレッドシート上の会社リストから情報を抽出、パーソナライズされた営業メールを Gmail で自動送信するツールです。
Python 版と比較して、環境構築が不要で Google ドライブ上で完結する「手軽さ」が最大の特徴です。

## 🚀 導入のメリットと使い分け
- **即時運用開始**: 
  Google アカウントさえあれば、ブラウザのみで設定から実行まで可能。サーバー管理やローカル環境の構築コストがゼロです。
- **サブツールとしての最適解**: 
  1日の送信上限（無料版：100通、Workspace：2,000通）を考慮し、大規模な一括送信ではなく「毎日50件ずつ丁寧に行うアプローチ」に最適化。
- **スプレッドシートとの親和性**: 
  共有機能により、チーム全員がリアルタイムで送信ステータスやリストの進捗を確認可能です。

## 🛠 機能概要
- **リスト自動読み込み**: A列（会社名）、B列（氏名）、C列（メール）を自動抽出。
- **送信ステータス管理**: 送信完了後にスプレッドシートへ「済」を自動入力。
- **エラー処理**: 特定の行でエラーが発生しても処理を中断せず、ログを記録して継続。

## 🛠 使用技術
- **言語**: Google Apps Script (JavaScript)
- **サービス**: 
  - SpreadsheetApp (データ取得・更新)
  - GmailApp (メール送信)

## 📝 注意事項
