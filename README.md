# 🚀 Business Automation Portfolio: 営業×ITで現場を加速させる

「営業現場の『負』を、テクノロジーで解決する。」
現場感覚を持ったエンジニア（Techセールス）として、売上向上と業務効率化に直結する自動化ツールを公開しています。

---

## 👤 自己紹介
**営業現場の課題をITで解決する「Techセールス」です。**
単にコードを書くのではなく、「そのツールを導入することで何時間削減できるか」「どれだけ有効商談が増えるか」というビジネス視点での開発を信条としています。

## 💡 スキルセット
- **Languages**: Python, Google Apps Script (GAS), VBA, SQL
- **Tools**: Google Workspace, Microsoft 365, X API, Discord API
- **Focus**: 業務プロセス自動化 (BPA), セールス・イネーブルメント, 顧客データ活用

## 🎯 開発への想い
テクノロジーは手段であり、目的は**「人がより創造的で、価値の高い仕事に集中できる環境を作ること」**です。
現場で働く人々がルーチンワークから解放され、顧客との対話や戦略策定に時間を使えるよう、現場に寄り添ったシステム開発を追求し続けます。
